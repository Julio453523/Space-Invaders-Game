# 🚀 Space Invader Game

A classic Space Invader-style arcade game built using Python & Pygame.

This project demonstrates game logic, object-oriented programming, and real-time user interaction.

## ✨ Features

- Player-controlled spaceship
- Enemy wave spawning system
- Collision detection
- Score tracking system
- Increasing difficulty over time
- Keyboard controls for movement and shooting

## 🛠 Tech Stack

- Language: Python 
- Graphics: Pygame 
- IDE: Visual Studio Code
